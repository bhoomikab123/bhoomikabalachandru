simple  calculator using AWt:
 Features:
Number buttons 0–9
Operations: + − × ÷
= to calculate results
C to clear the screen
Smooth input handling using ActionListener


How it works:
Click numbers → choose an operator → enter second number → hit = to get the result.
Everything is displayed in the top text field.

Built With:
Java AWT (Frame, Button, TextField)
Event handling with ActionListener

Shows how many attempts you used
New Game button to restart anytime

 Works With
Java AWT buttons, labels, text fields, and event handling.
